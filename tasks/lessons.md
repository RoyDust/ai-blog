# Lessons

- 2026-05-22: 文章详情路由切换里的头部宽度变化发生在 `.reader-nav`，只给主体 `.reader-layout-frame` 加宽度过渡不会影响导航头部。命名 View Transition 的具体 `old/new` 规则要放在通配 `old/new(*)` 后面，避免慢速时长被后写的通配规则覆盖。
- 2026-05-22: 文章详情右侧目录的动效语义应是同一位置的内容重载，不是空间进入。目录 rail 不应使用 `x` 方向滑入；优先保留卡片位置，用骨架态到内容态的纵向 layout 高度变化表达加载完成。
- 2026-05-30: 后台治理类列表不能用接口 `take` 固定截断后再前端内容分页；评论、文章、标签这类会增长的数据必须由接口返回 `pagination` 与真实统计，前端只渲染当前页。
